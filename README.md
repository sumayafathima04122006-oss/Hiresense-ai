# HireSense AI

HireSense AI is an AI-powered interview preparation platform that simulates real interview scenarios and provides structured feedback on user responses.

## 🚀 Features
- AI-generated interview questions based on job roles
- Answer evaluation using GPT
- Feedback on clarity, relevance, and communication
- (Planned) Voice input and speech analysis
- (Planned) Facial expression analysis

## 🧠 Tech Stack
- No-code platform: Lovable
- AI: OpenAI GPT API
- Speech-to-text: Whisper / AssemblyAI (planned)
- Workflow: Make / Zapier (if needed)

## ⚙️ How It Works
1. User selects a job role  
2. AI generates interview questions  
3. User submits answers (text/voice)  
4. AI evaluates response and provides feedback  
5. Results are stored for progress tracking
